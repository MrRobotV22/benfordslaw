Installation
################


Pypi
**********************

.. code-block:: console

    # Install from Pypi:
    pip install benfordslaw

    # Force update to latest version
    pip install -U benfordslaw


Github source
************************************

.. code-block:: console

    # Install directly from github
    pip install git+https://github.com/erdogant/benfordslaw


Uninstalling
################

Note that the removal of the environment will also remove the ``benfordslaw`` installation.

.. code-block:: console

    # Install from Pypi:
    pip uninstall benfordslaw
